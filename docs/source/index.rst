Development
===========

Create virtual environment and install dependencies::

    git@github.com:QubitPi/wilhelm-graphdb-python.git
    cd wilhelm-graphdb-python
    python3 -m venv .venv
    . .venv/bin/activate
    pip3 install -r requirements.txt
